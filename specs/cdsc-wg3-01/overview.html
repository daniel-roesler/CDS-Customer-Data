Customer Data - Overview

This is a summary overview of cdsc-wg3-01, which describes how companies and technical users (‚ÄúClients‚Äù) can request access to customer data from utilities and other centralized entities (‚ÄúServers‚Äù). This specification builds upon cdsc-wg1-02, which allows Clients to register with a Server and be granted access to request various scopes of data from that Server using the OAuth protocol.

For example, this specification describes how an energy efficiency app can request access from a utility customer to see their account number, rate plan, and last year of energy usage.
